Google AI

人工智能研究与应用

Google AI是人工智能领域的领导者，在机器学习、深度学习、计算机视觉等方面有大量研究成果和开源工具。Google AI的研究涵盖了从基础算法到实际应用的各个方面。
官方网站 AI发现 研究 机器学习指南 Google Cloud AI

Microsoft AI

人工智能平台与工具

Microsoft AI提供了完整的人工智能平台和工具，包括Azure AI、认知服务等，支持企业级AI应用开发。Microsoft AI致力于使AI技术更加普及和易于使用。
官方网站 Azure AI AI学习 Microsoft Research AI AI系统 GitHub

TensorFlow

深度学习框架

TensorFlow是Google开发的开源深度学习框架，广泛应用于机器学习研究和生产环境，是AI开发的重要工具。TensorFlow支持多种平台和设备，包括CPU、GPU和TPU。
官方网站 教程 API文档 GitHub TensorFlow Lite

PyTorch

动态深度学习框架

PyTorch是Facebook开发的开源深度学习框架，以其动态计算图和易用性而闻名，广泛应用于研究和生产环境。PyTorch在研究社区中特别受欢迎。
官方网站 教程 API文档 GitHub TorchVision

scikit-learn

机器学习库

scikit-learn是Python的开源机器学习库，提供了简单高效的数据挖掘和数据分析工具，适合各种机器学习任务，包括分类、回归、聚类等。
官方网站 教程 API文档 GitHub 示例

Keras

高级神经网络API

Keras是一个高级神经网络API，运行在TensorFlow、Theano或CNTK之上，以其用户友好性、模块化和可扩展性而闻名。Keras使深度学习模型的开发更加简单。
官方网站 指南 API文档 GitHub 示例
